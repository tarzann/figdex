## 📊 עלות למשתמש Pro (שימוש ממוצע)

### תרחיש שימוש ממוצע:
- **5 אינדקסים ליום**
- **500 פריימים לאינדקס (ממוצע)**
- **2,500 פריימים ליום**
- **75,000 פריימים בחודש**

### חישוב עלויות Vercel:

#### Function Executions:
- 5 אינדקסים × 250 chunks (2 פריימים לכל chunk) = 1,250 executions/יום
- 37,500 executions/חודש
- **GB-hours:** 1,250 × 15 שניות × 2GB = 37,500 GB-seconds = 10.4 GB-hours/יום
- **עלות GB-hours:** 10.4 × 30 × $0.0000166667 = **$5.20/חודש**

#### Bandwidth:
- 2,500 פריימים × 200KB = 500MB/יום = 15GB/חודש
- **בחבילת Pro:** ✅ נכלל (1TB/חודש)

**עלות Vercel למשתמש:** $5.20/חודש (GB-hours בלבד, הבסיס $20 משותף)

### חישוב עלויות Supabase:

#### Database size:
- 75,000 פריימים × 5KB metadata = 375MB
- **בחבילת Pro:** ✅ נכלל (8GB)

#### File storage:
- 75,000 פריימים × 200KB = 15GB
- **בחבילת Pro:** ✅ נכלל (100GB)

#### API requests:
- 37,500 executions × 5 queries = 187,500 queries/חודש
- **בחבילת Pro:** ✅ נכלל (5 מיליון/חודש)

#### Database reads/writes:
- **בחבילת Pro:** ✅ נכלל

#### Bandwidth:
- 15GB uploads/חודש
- **בחבילת Pro:** ✅ נכלל (250GB/חודש)

**עלות Supabase למשתמש:** $0/חודש (כלול בחבילת Pro הבסיסית)

## 📈 חישוב קיבולת

### נוסחת רווחיות:

```
רווח = הכנסות - עלויות

רווח = (מספר משתמשים × $15) - ($45 תשתית + מספר משתמשים × $5.20)
```

### נקודת שוויון (רווח = 0):

```
0 = (N × $15) - ($45 + N × $5.20)
0 = $15N - $45 - $5.20N
$45 = $15N - $5.20N
$45 = $9.80N
N = $45 / $9.80
N = 4.59 משתמשים
```

### עם הגבלות חודשיות (300 אינדקסים, 30,000 פריימים):

בהתאם להגבלות החודשיות שהגדרנו, משתמש Pro מוגבל ל:
- 300 אינדקסים/חודש
- 30,000 פריימים/חודש

זה מקטין את השימוש הממוצע:
- 300 אינדקסים/חודש = 10 אינדקסים/יום
- 30,000 פריימים/חודש = ~100 פריימים/אינדקס

**עלות GB-hours חדשה:**
- 10 אינדקסים × 50 chunks = 500 executions/יום
- 15,000 executions/חודש
- **GB-hours:** 500 × 15s × 2GB = 15,000 GB-seconds = 4.17 GB-hours/יום
- **עלות GB-hours:** 4.17 × 30 × $0.0000166667 = **$2.08/חודש**

**חישוב חדש:**
```
0 = (N × $15) - ($45 + N × $2.08)
0 = $15N - $45 - $2.08N
$45 = $12.92N
N = $45 / $12.92
N = 3.48 משתמשים
```
