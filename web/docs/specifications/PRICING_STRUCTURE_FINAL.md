**For designers working on real projects**

**Price: $29 / month**

- **1,000 frames quota** (permanent)

- **Private galleries**
- **Standard processing priority**

### Credits Included:

- **500 credits / month** (renews monthly)

### 💳 What Credits Can Buy:

**Indexing:**
- **1 file index** = 50 credits
- **1 file re-index** = 50 credits

**Quota Increases (monthly):**
- **+1 file quota** = 100 credits / month
- **+1,000 frames quota** = 100 credits / month

**Additional Re-indexes:**
- **+1 re-index** = 50 credits (one-time)

### 💰 Purchase Additional Credits:

- **100 credits** = $5 (one-time or monthly subscription)
- **250 credits** = $10 (one-time or monthly subscription)
- **500 credits** = $18 (one-time or monthly subscription)
- **1,000 credits** = $35 (one-time or monthly subscription)

## 📊 Economic Validation

### Pro Plan ($29/month)

**Base Cost:**
- Infrastructure: $45/month (shared)
- Variable per user: ~$2/month

**Break-even:** 2 users (2 × $29 = $58 > $47 cost)

**With 5 users:**
- Revenue: $145/month
- Cost: $55/month
- **Profit: $90/month (62% margin)** ✅

---

### Team Plan ($49/month)

**Base Cost:**
- Infrastructure: $45/month (shared)
- Variable per user: ~$3/month

**Break-even:** 1 user (but assumes 2+ Pro users sharing infrastructure)

**With 5 teams:**
- Revenue: $245/month
- Cost: $60/month
- **Profit: $185/month (75% margin)** ✅
