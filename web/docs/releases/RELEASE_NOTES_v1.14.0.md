## 🎯 מה חדש בגרסה זו?

### ✨ תכונות חדשות

#### 1. סינון פריימים מוסתרים
המערכת כעת מתעלמת אוטומטית מפריימים מוסתרים (`visible === false`) בכל שלבי האינדקס:
- ספירת פריימים
- יצירת job
- עיבוד job

**יתרון:** אינדקס נקי יותר, ללא פריימים לא רלוונטיים.

---

### 🚀 שיפורי ביצועים

#### 1. הפחתת פניות לשרתים ב-~50%
- **Polling Intervals:**
  - Status checks: 15s → **30s**
  - Job processing: 10s → **20s**
  - Error retry: 30s → **60s**

**תוצאה:**
- ~50% פחות קריאות ל-Vercel
- ~70% פחות עדכונים ל-Supabase
- ~50% פחות קריאות ל-Figma API

#### 2. הפחתת עדכוני Database ב-~70%
עדכון רק על שינויים משמעותיים:
- שינוי status
- שינוי progress > 5%
- שינוי error
- שינוי indexId

**תוצאה:** פחות עומס על Supabase, פחות עלויות.

---

### 🐛 תיקוני באגים

#### 1. תיקון שגיאת React #418
- **בעיה:** שגיאת React hooks בגלל dependencies לא נכונים
- **פתרון:** הוספת `useCallback` ותיקון dependencies
- **תוצאה:** אין יותר שגיאות React

#### 2. תיקון עצירת Job אוטומטית
Job עוצר אוטומטית במקרה של שגיאות קריטיות:
- שגיאות אימות (401, 403)
- שגיאות בעיבוד דף
- שגיאות קריטיות ב-upload
- שגיאות קריטיות ב-Figma API

**יתרון:** אין יותר jobs תקועים.

#### 3. תיקון בעיית ERR_INSUFFICIENT_RESOURCES
- **בעיה:** יותר מדי קריאות מקבילות
- **פתרון:** מניעת קריאות מקבילות + הגדלת מרווחים
- **תוצאה:** מערכת יציבה יותר

#### 4. תיקון בעיית Job Completion המוקדמת
- **בעיה:** Jobs הושלמו לפני שכל הפריימים עובדו
- **פתרון:** מעקב מפורט אחרי frames per page
- **תוצאה:** כל הפריימים מעובדים כעת
