אני אתאר לך את הפלאגין בפירוט כדי שתוכל לבנות אותו מחדש:

## **Frames Index Generator - תיאור מפורט**

### **מטרה עיקרית:**
הפלאגין יוצר **אינדקס ויזואלי** של כל הפריימים (frames) בפרויקט פיגמה, ומציג אותם בעמוד מיוחד עם תמונות ממוזערות וקישורים ישירים.

### **מבנה הקבצים הנדרש:**

#### **1. `manifest.json`**
```json
{
  "name": "Frames Index Generator",
  "id": "1520431284479609393",
  "api": "1.0.0",
  "ui": "ui.html",
  "main": "code.js",
  "capabilities": [],
  "enableProposedApi": false,
  "documentAccess": "dynamic-page",
  "editorType": ["figma"],
  "networkAccess": {
    "allowedDomains": ["none"]
  }
}
```

#### **2. `ui.html` - ממשק המשתמש**
**מבנה הממשק:**
- **Index Marking Tools** (בראש הממשק)
  - כפתור "Mark selected frames as NO-INDEX"
  - כפתור "Mark all frames as INDEX"
- **Pages List** (רשימת עמודים)
  - checkbox ראשי לבחירת כל העמודים
  - רשימת עמודים עם checkboxes
  - כפתור רענון (🔄)
- **Create Index** (כפתור יצירת אינדקס)
- **Progress Bar** (סרגל התקדמות)
- **Status Messages** (הודעות סטטוס)

**פונקציונליות UI:**
- הצגת עמודים עם תמיכה במבנה תיקיות
- עמודים בשם "Frame-Index" מוצגים בראש הרשימה ומוגדרים כ-disabled
- בחירה מרובה של עמודים
- הצגת התקדמות בזמן אמת
- הודעות הצלחה/שגיאה

#### **3. `code.js` - הלוגיקה הראשית**

**פונקציות עיקריות:**

**א. `findAllFrames(node)`**
- פונקציה רקורסיבית למציאת כל הפריימים בעמוד
- מחזירה מערך של ID של פריימים

**ב. `findAllTexts(node)`**
- פונקציה רקורסיבית למציאת כל הטקסטים בפריים
- מחזירה מערך של טקסטים

**ג. `createFramesIndexPage(selectedPageIds)`**
- הפונקציה הראשית ליצירת האינדקס
- יוצרת עמוד חדש בשם "􀧌 Frames Index"
- מייצא תמונות ממוזערות של כל הפריימים
- יוצר קישורים ישירים לפריימים המקוריים

**הודעות שמטפל בהן:**
- `check-selection` - בדיקת בחירה בפיגמה
- `mark-no-index` - סימון פריימים כ-NO_INDEX
- `mark-all-index` - הסרת סימון NO_INDEX
- `refresh-pages` - רענון רשימת עמודים
- `start-advanced` - יצירת אינדקס

### **זרימת העבודה:**

#### **1. טעינת הפלאגין:**
- טוען את כל העמודים
- מסנן עמודים ריקים ומפרידים
- מזהה עמודים בשם "Frame-Index"
- שולח רשימת עמודים ל-UI

#### **2. בחירת עמודים:**
- המשתמש בוחר עמודים מהרשימה
- עמודים בשם "Frame-Index" מוגדרים כ-disabled
- כפתור "Create Index" מופעל רק אם יש עמודים נבחרים

#### **3. יצירת אינדקס:**
- יוצר עמוד חדש בשם "􀧌 Frames Index"
- סורק את כל הפריימים בעמודים הנבחרים
- מייצא תמונות ממוזערות (50% מגודל מקורי)
- יוצר פריימים קטנים עם תמונות ותוויות
- מוסיף קישורים ישירים לפריימים המקוריים

#### **4. כלי סימון:**
- **Mark NO-INDEX:** מוסיף "[NO_INDEX]" לשם הפריים
- **Mark INDEX:** מסיר "[NO_INDEX]" מכל הפריימים
- פריימים עם "[NO_INDEX]" לא יופיעו באינדקס

### **תכונות מיוחדות:**

#### **1. תמיכה במבנה תיקיות:**
- מזהה עמודים שמתפקדים כתיקיות (ללא פריימים)
- מציג היררכיה: "תיקיה -> עמוד"

#### **2. עמוד אינדקס:**
- עמודים בשם "Frame-Index" מוצגים בראש הרשימה
- מוגדרים כ-disabled ולא ניתן לבחור אותם
- מוצגים בצבע כתום עם התוספת "(Index Page)"

#### **3. קישורים ישירים:**
- כל תמונה ממוזערת מקושרת ישירות לפריים המקורי
- לחיצה על התמונה תעביר לפריים בפיגמה

#### **4. שמירת טקסטים:**
- שומר את כל הטקסטים מכל פריים כטקסט מוסתר
- מאפשר חיפוש וניתוח תוכן

#### **5. התקדמות בזמן אמת:**
- מציג אחוזי התקדמות בעת יצירת האינדקס
- הערכת זמן נותר
